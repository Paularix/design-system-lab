# 🎨 Design System Lab
### A living laboratory of components • By Paula Marcos

Welcome to my personal playground. This design system is a **constantly evolving project** where I experiment with new ideas, test complex interactions, and craft the components that eventually power my portfolio.

---

## 🛠️ Core Stack
* **Material UI (MUI):** The robust foundation for all components.
* **Framer Motion:** The engine behind the smooth, high-performance animations I love to build.
* **TypeScript:** Keeping the code clean and scalable as the library grows.
* **Storybook:** My interactive stage for testing how components behave in isolation.
