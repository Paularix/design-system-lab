# Design System Lab 🎨

A curated collection of interactive React components built with **TypeScript**, **MUI**, and **Framer Motion**.

## What is it for?
This library is designed to provide reusable, high-quality UI components (like interactive cards, menus, and animated buttons) to ensure visual consistency across different React applications.

## Installation

Since this is a private repository, install it directly from GitHub:

```bash
npm install git+[https://github.com/Paularix/design-system-lab.git](https://github.com/Paularix/design-system-lab.git)
